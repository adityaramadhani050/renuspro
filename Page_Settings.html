WA Bot Notifikasi

Kirim notifikasi otomatis ke grup WhatsApp via Baileys

Notifikasi dikirim langsung ke grup WhatsApp via server Baileys Anda. Pastikan server sedang berjalan dan WA sudah terhubung.

URL publik server Baileys tanpa trailing slash. Pesan dikirim ke /api/messages/send.

JID grup WhatsApp tujuan, format xxxxxxxxxx-xxxxxxxxxx@g.us. Cek di dashboard Baileys → Conversations.

Event yang akan memicu notifikasi:

Invoice Dibuat — saat invoice baru diterbitkan
Invoice Lunas — saat status invoice diubah ke Lunas
Work Order Dibuat — saat penawaran ditandai Deal
Request Invoice — saat Sales mengajukan request buat invoice
