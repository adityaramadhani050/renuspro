WA Bot Notifikasi

Kirim notifikasi otomatis via Meta Cloud API (WhatsApp Business)

Gunakan Access Token dari System User di Meta Business Manager. Token dari App Dashboard (temporary) tidak disarankan untuk produksi.

Meta Developer → App → WhatsApp → API Setup → Phone Number ID (bukan nomor HP-nya).

Format internasional tanpa tanda + (contoh: 628123456789). Notifikasi dikirim ke nomor ini.

Event yang akan memicu notifikasi:

Invoice Dibuat — saat invoice baru diterbitkan
Invoice Lunas — saat status invoice diubah ke Lunas
Work Order Dibuat — saat penawaran ditandai Deal
Request Invoice — saat Sales mengajukan request buat invoice
